What is Card in Flutter?
In Flutter, a Card is a material design container that:
Has rounded corners
Has shadow (elevation)
Has background color
Usually contains content like text, icons, images
🔹 Important Properties
Property	Purpose
color	Background color of card
elevation	Shadow depth (higher = stronger shadow)
child	What goes inside the card
margin	Space outside the card
